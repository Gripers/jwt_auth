# JWT Аутентификация

## Как пользоваться?

```
1. git clone https://github.com/Gripers/jwt_auth.git
2. cd jwt_auth
3. команда: npm install
4. команда: npm run server
5. хост(ссылка на http запросы): http://localhost:8080/
```
